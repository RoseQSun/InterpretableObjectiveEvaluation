# Conda Cheatsheet

## Create Environment

```bash
conda create -n env_name python=3.10
```

## Activate Environment

```bash
conda activate env_name
```

## Install Packages

```bash
pip install package_name_1 package_name_2
```

Install jupyter lab

```bash
pip install jupyterlab
```

## Remove Environment

```bash
conda env remove -n env_name
```
